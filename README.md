Pomodoro-Sublime-Plugin
=========================


Usage: 
-----------------------------
to start run command `pomodoro: start`
to cancel a pomodoro run command `pomodoro: cancel`
  
  
`Preferences->Package Settings->Sublime-Pomodoro` you can modify the following values:
```
{
    "pomodoro_mins": 25,
    "shortbreak_mins": 5,
    "longbreak_mins": 10,
    "number_pomodoro": 4
}
```
